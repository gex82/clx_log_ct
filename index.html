Autopilot demo didn’t start

This usually means the JavaScript bundle didn’t load (or the build output directory is misconfigured).

  • Cloudflare Pages: set Build command to npm run build and Output directory to dist.
  • Open DevTools Console/Network and look for a 404 on /assets/* or a MIME-type error.
  • If you deployed under a subpath, set VITE_ROUTER_BASENAME (see README).
